# Contributing to Circle to Search

First off, thank you for considering contributing to Circle to Search!

## Code of Conduct

By participating in this project, you are expected to uphold general open-source standards of kindness, collaboration, and constructive feedback.

## How Can I Contribute?

### Reporting Bugs
If you find a bug:
- Ensure the bug was not already reported by searching through issues.
- If you're unable to find an open issue addressing the problem, open a new one. Be sure to include a title and clear description, with step-by-step reproduction instructions.

### Suggesting Enhancements
Enhancements can range from new features to UI polish:
- To submit a suggestion, provide a clear description of the feature and explain how it improves the application.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added new code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (if available).
5. Submit your Pull Request with a clear description of the problem and detailed execution of your solution.

Thank you!
